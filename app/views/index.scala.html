Block Puzzle Regeln

Block Puzzle ist ein Spiel basierend auf dem Spiel Tetris.

Es werden jeweils 3 Blöcke angezeigt, welche in einem Spielfeld mit 8x8 Feldern platziert werden müssen. Ziel des Spiels ist es Reihen oder Spalten durchgehend mit den Blöcken zu füllen.

Sobald eine Reihe oder Spalte gefüllt ist verschwindet sie, wie bei Tetris.

Das Spiel ist ein Endlosspiel, es gibt kein natürliches Ende. Für das plazieren der Blöcke kann man sich so viel Zeit nehmen wie man will.

Beendet ist das Spiel entweder wenn keine Lust mehr hat oder wenn keine Blöcke im Spielfeld mehr platziert werden können

Man startet mit einem leeren Spielfeld und 3 zufällig generierten Blöcken:

Danach kann man die Blöcke frei auf dem Spielfeld verteilen, wenn alle 3 Blöcke verteilt wurden, entstehen 3 neue Blöcke.
